# Описание экранов Mini App

## 1. Главная (Dashboard)

**Путь:** `/`

### Элементы:
- Карточка подписки
  - Статус (Активна / Истекает / Неактивна)
  - Название тарифа
  - Дата окончания
  - Прогресс времени
  - Прогресс трафика
- Метрики (3 карточки)
  - Использовано трафика
  - Осталось дней
  - Автопродление (Вкл/Выкл)
- Быстрые действия
  - Скопировать конфиг
  - Показать QR-код
  - Пригласить друга
- Статистика за 7 дней

### API:
```
GET /api/webapp/dashboard
```

---

## 2. Лояльность

**Путь:** `/loyalty`

### Элементы:
- Карточка статуса (Bronze/Silver/Gold/Platinum)
- Текущие баллы
- Прогресс до следующего уровня
- Текущая скидка
- Таблица цен со скидками
- История начислений

### API:
```
GET /api/webapp/loyalty
```

---

## 3. Магазин

**Путь:** `/shop`

### Элементы:
- Карточки тарифов (1/3/6/12 мес)
  - Базовая цена (зачёркнута)
  - Цена со скидкой
  - Экономия в %
- Переключатель "Для себя / В подарок"
- Выбор способа оплаты

### API:
```
GET /api/webapp/plans
POST /api/webapp/payment/create
```

---

## 4. Подарки

**Путь:** `/gifts`

### Элементы:
- Поле активации кода
- Мои подарки (купленные)
- Полученные подарки

### API:
```
GET /api/webapp/gifts
POST /api/webapp/gifts/activate
```

---

## 5. История

**Путь:** `/history`

### Элементы:
- Список транзакций
- Фильтры по периоду

### API:
```
GET /api/webapp/payments
```

---

## 6. Помощь

**Путь:** `/help`

### Элементы:
- FAQ (аккордеон)
- Инструкция по подключению
- Кнопка "Написать в поддержку"
- Ссылки на соцсети
